语言

高级语言需要翻译成机器语言，翻译方式：

  • 编译 不可移植，C/C++
  • 解释 运行一行执行一行

Python 是解释型高级动态编程语言

动态编程语言：运行时可改变其结构，Python/JavaScript

  • 强类型 不容忍隐性类型转换

  • 弱类型 容忍隐性类型转换

  • 静态类型 编译时就知道每个变量的类型，类型错误是语法错误

  • 动态类型 类型错误是运行错误

  • 命令式编程 面向计算机硬件的抽象，程序是冯诺依曼机的指令序列
  • 函数式编程 面向数字，计算是表达式求值，程序是表达式
